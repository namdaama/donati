## SVG書き出し方法（Illustrator）

### 簡単な方法:
1. 書き出したいオブジェクトを選択
2. 「ファイル」→「書き出し」→「スクリーン用に書き出し」
3. 形式: **SVG**
4. 設定:
   - スタイル: プレゼンテーション属性
   - フォント: SVG（または アウトライン化）
   - 画像: リンク
   - オブジェクトID: レイヤー名
   - 小数点: 2
   - 縮小: チェック
   - レスポンシブ: チェック
5. 「アセットを書き出し」をクリック

### もっと簡単な方法:
各要素を選択 → 右クリック → 「書き出し形式」→ SVG

※設定が分からない場合は、デフォルト設定で大丈夫です！
