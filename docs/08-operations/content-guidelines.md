# コンテンツ文言ガイドライン

DONATIサイト全体で採用する「柔らかく親しみやすい表現」の文言ガイド。Issue #19実装済み。

## 基本方針

### 1. 親しみやすさ
硬い表現を避け、温かみのある言葉選びを心がける。

### 2. わかりやすさ
専門用語は最小限にし、誰にでも理解しやすい表現を使用。

### 3. ワクワク感
科学の楽しさ・興味深さが伝わる表現を重視。

### 4. 安心感
気軽に参加・問い合わせできる雰囲気作りを意識。

## 文言例

### ボタンテキスト
| 硬い表現 | 親しみやすい表現 |
|---------|----------------|
| お申し込みはこちら | 参加してみる |
| 詳細を確認する | もっと知りたい |
| ご依頼ください | お気軽にお問い合わせ |

### サービス紹介
| 硬い表現 | 親しみやすい表現 |
|---------|----------------|
| 提供いたします | お届けします |
| 実施いたします | ご一緒に楽しみます |
| 対応可能です | やってみることができます |

### お問い合わせ・相談
| 硬い表現 | 親しみやすい表現 |
|---------|----------------|
| ご不明な点は | 気になることがあれば |
| お気軽にお問い合わせください | 何でもお聞きください |
| 対応させていただきます | お手伝いします |

### 実績紹介
| 硬い表現 | 親しみやすい表現 |
|---------|----------------|
| 実施しました | みんなで楽しみました |
| 開催いたしました | 開いてみました |
| 提供させていただきました | シェアしてみました |

## 文言適用範囲

### 適用対象
- ボタンテキスト
- セクションタイトル
- 説明文・キャッチコピー
- フォームラベル
- エラーメッセージ
- 404ページ

### 避けるべき表現
- 「いたします」「いたただきます」（敬語の過剰使用）
- 「対応可能」「実施」（ビジネス用語）
- 「必須」「入力必須」（堅い指示）
- 「エラー」（ネガティブ）

### 推奨される表現パターン
- 「～してみる」（チャレンジ感）
- 「～できます」（ポジティブ）
- 「～をお手伝い」（親近感）
- 「～してみませんか」（提案）

## ページ別実装例

### トップページ（index）
```
❌ 科学実験サービスの提供を行っております
✅ 科学の面白さを一緒に探ってみませんか
```

### サービス紹介（services）
```
❌ 対応可能なサービスは以下の通りです
✅ こんなことができます
```

### お問い合わせ（contact）
```
❌ お問い合わせフォームです。必須項目を入力してください
✅ ご質問・ご相談があれば、お気軽にどうぞ
```

### 404ページ
```
❌ ページが見つかりません
✅ あれ、このページはなくなっちゃったみたい
```

## 継続的な改善

### 新規コンテンツ作成時
- 本ガイドラインに基づいた文言を使用
- 硬い表現が混入していないか確認
- 読み手の視点に立って評価

### ユーザーフィードバック対応
- コンテンツ改善の提案があれば、本ガイドラインに反映
- クライアント（フジ）の意見を優先

### 科学専門性とのバランス
- 科学の専門性は保ちながら、わかりやすい表現を心がける
- 難しい概念は「～という流れです」など説明的に表現

## チェックリスト

新しいテキストを追加する際は、以下の点を確認:

- [ ] 硬い敬語を使っていないか
- [ ] ビジネス用語を避けているか
- [ ] 「～してみる」「～できます」など前向きな表現か
- [ ] 読み手を気軽に感じさせるか
- [ ] 科学の面白さが伝わるか
- [ ] クライアントの意図が反映されているか
