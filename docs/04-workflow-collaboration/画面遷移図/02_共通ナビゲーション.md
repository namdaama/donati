## 問題点と改善提案

### 1. フッター「依頼の流れ」のリンク先

**現状:**
```astro
<a href="/services">・依頼の流れ</a>
```

**問題点:**
- `/services`ページのトップに飛ぶため、ユーザーが「ご依頼の流れ」セクションまでスクロールする必要がある
- 他のアンカーリンク（`/services#faq`）と一貫性がない

**推奨される修正:**
```astro
<a href="/services#request-flow">・依頼の流れ</a>
```

**参照**: [サービスページ詳細図](./04_サービスページ詳細.md)の `#request-flow` セクション

---

### 2. Header「DONATIとは？」と「私たちについて」の統合

**現状:**
- 「DONATIとは？」 → `/about`
- 「私たちについて」 → `/about`

**問題点:**
- 両方とも同じ `/about` ページへリンクしているが、内容が未定義
- ユーザーにとって2つの違いが不明確

**提案A: 統合**
- `/about`ページに両方の内容を含める（ミッション + スタッフ紹介）
- Headerのアイコンは1つに統合

**提案B: 分離**
- `/about` → ミッション/ビジョン/サイエンスコミュニケーションへの思い
- `/staff` → スタッフ個人の紹介（趣味、好きな実験、メッセージ等）

---

### 3. Footer「私たちについて」のリンク先

**現状:**
```astro
<a href="/staff">・私たちについて</a>
```

**問題点:**
- Headerは `/about`、Footerは `/staff` で不一致
- どちらも未実装のため、統一の機会

**推奨される修正:**
- ヘッダーと統一して `/about` にするか、`/staff` を独立実装するかを決定
