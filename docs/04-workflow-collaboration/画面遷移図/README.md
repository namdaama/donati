## 💡 Mermaidダイアグラムの閲覧方法

### 方法1: GitHub上で表示（最も簡単）
1. GitHubリポジトリ上でMarkdownファイルを開く
2. Mermaidが自動レンダリングされます

### 方法2: VSCode拡張機能
1. VSCodeで「Markdown Preview Mermaid Support」拡張機能をインストール
2. Markdownファイルを開く
3. `Ctrl+Shift+V`（Windows）でプレビュー表示

### 方法3: Mermaid Live Editor（オンライン）
1. https://mermaid.live/ にアクセス
2. Mermaidコードブロックの中身をコピー
3. 左側のエディタに貼り付け → 右側に図が表示
4. PNG/SVG形式でエクスポート可能（Figmaに読み込み可能）
