# 要件定義: Footer文言修正とリンク再設定 (Issue #133)

## 目的
Footerコンポーネントの文言を更新し、リンク構造を整理する

## 現状把握

### 現在のFooter構成（3カラム）

**左側 - 会社情報**
- タイトル: 「サイエンス＆スペース ラボ」「DONATI」
- キャッチコピー: 「科学の楽しさを伝え、世界をワクワクでいっぱいにする」

**中央 - メニュー**
- ・DONATIとは？ → /about
- ・私たちについて → /about
- ・お知らせ（イベント案内など）→ /news
- ・サービス内容について → /services
- ・依頼の流れ → /services
- ・活動経歴 → /professional-experience

**右側 - お問合せ**
- 見出し: 「お問合せ」
- 説明文: 「時間・予算・内容、ご相談OKです！組み合わせ内容も柔軟に対応します！」
- リンクなし

**最下部**
- コピーライト: 「© サイエンス&スペース ラボ DONATI」

## 要求される変更内容

### 左側（変更なし）
- サイエンス＆スペースラボ DONATI
- 科学の楽しさを伝え、世界をワクワクでいっぱいにする

### 中央（メニュー項目の変更）
変更前 → 変更後:
- ・DONATIとは？ → ・サイエンス事業
- ・私たちについて → ・星空事業
- ・お知らせ（イベント案内など）→ ・実績
- ・サービス内容について → ・料金・FAQ
- ・依頼の流れ → ・私たちについて
- ・活動経歴 → ・お知らせ

### 右側（お問合せセクションの変更）
- 見出し: 「お問合せ」→ 「お問合せ（リンク張ってください）」
- 説明文: 「実施内容・時間・ご予算について、ご希望に合わせてご相談を承ります。まずはお気軽にご連絡ください。」
- **お問合せページへのリンクを追加**

### 最下部（コピーライトの変更）
- 「© サイエンス&スペース ラボ DONATI」
- → 「© 2026 サイエンス＆スペースラボ DONATI (Science & Space Lab DONATI) All rights reserved.」

## リンク先の確認が必要な項目

以下のメニュー項目について、適切なリンク先を決定する必要があります:

1. **サイエンス事業** → 現在は/aboutだが、/servicesや専用ページの可能性
2. **星空事業** → 新しいリンク先が必要（現在該当ページなし？）
3. **実績** → /professional-experienceが適切
4. **料金・FAQ** → /servicesまたは専用ページ
5. **私たちについて** → /about
6. **お知らせ** → /news（既存）
7. **お問合せ** → /contact
